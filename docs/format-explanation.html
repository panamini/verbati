Two A4 adaptations of the classical page canon

A proportional comparison of two Tschichold-inspired layout models for the A4 sheet, drawn with scale-consistent SVG diagrams, measured margins, captions, and a comparative editorial table.

Classical canon: 2 : 3 : 4 : 6 = 1 : 1.5 : 2 : 3

These A4 versions are adaptations, not exact canon. The historical construction assumes a 2:3 page proportion, while A4 is fixed at 1:√2. To preserve the spirit of the inner / top / outer / bottom hierarchy on ISO paper, the sequence shifts toward 1 : 1.414 : 2 : 2.828.

Context

A4 is not the 2:3 page assumed by the historical canon, so these are proportional translations rather than exact reconstructions.

Comparison

Same adapted sequence, different text-area density

Reading the difference

The 9-division version feels more ceremonial and open, with stronger lower and outer whitespace. The 12-division version yields a more efficient editorial page with a larger usable block while preserving the same directional hierarchy. Both retain the adapted A4 margin sequence 1 : 1.414 : 2 : 2.828.
